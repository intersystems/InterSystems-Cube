InterSystems-cube by obiwanchinobi

A way to manage InterSystems' OSX installations.

I install dozens of InterSystems Cach√© and Ensemble builds per month and I wanted a better way of managing all of my installations, especially which ones should automatically start on system boot.

Dependencies

If you don't have InterSystems installed locally on OSX then don't waste your time. This has only been tested on Snow Leopard and Lion.

Install

Just download and move the app to your Applications directory. It will prompt you for a password the first time you use it because it is installing a script that is responsible for starting/stopping/restarting and managing StartupItems which all require root privileges.
