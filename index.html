InterSystems-cube
by Dave Tong (Obiwan Chinobi)

A way to manage InterSystems' OSX local-installations.

Built with the intent to help better manage all of your local InterSystems' Caché and Ensemble Instances.

Features

  • Start/Stop/Restart Instances
  • Autostart an Instance on system startup
  • Telnet into Caché/Ensemble
  • Open up Management Portal
  • Open up directory the Instance is installed to
  • Open up Documentation and Class Reference

Dependencies

If you don't have InterSystems technology installed locally on OSX then perhaps you should start.

This has only been tested on Snow Leopard and Lion.

Installation Instructions

  • Download and unzip App
  • Move App to your Applications directory
  • Run it - this will prompt you for a password the first time you run it because it is installing a Helper Job that requires root privileges. This Helper is used for:
    • Starting/stopping/restarting an Instance
    • Managing StartupItems - requires root privileges
